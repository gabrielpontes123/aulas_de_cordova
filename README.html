2 Repositório para Aulas De Cordova

Este projeto tem por finalidade prover subsidios àss aulas de mobile utilizando o Apache Cordova.

2.1 Como utilizar este repositório

2.1.1 faça o checkout

git clone https://github.com/wagnermarques/aulas_de_cordova.git

2.1.2 Apresentação dos projetos existentes

A idéia a aprensentar vários pequenos projetos pra ilustrar o funcionamento de cada um.

    2. Vamos fazer o build do projeto e depois rodá-lo

      Fazer o build significa criar o arquivo .apk que é um instalador do seu projeto mobile. Esse instalador vc pode depois passar pro seu celular e instalá-lo. Quando rodamos no emulador é exatamene a mesma coisa que é feito, ou seja, gerado o apk e depois instalá-lo no aparelho virtual.

      Bom então vc precisa primeiro buidar o seu projeto, vamos fazer isso? 

      cd /home/wagner/wagnerdocri@gmail.com3/envs/env-dev/sources/aulas_de_cordova/projetos/jquerymob_menu/
cordova build android

      Unable to load PlatformApi from platform. Error: Cannot find module '/home/wagner/wagnerdocri@gmail.com3/envs/env-dev/sources/aulas_de_cordova/projetos/jquerymob_menu/platforms/android/cordova/Api.js'

  2. Inclusão do Jquery Mobile

    Provavelmente o melhor lugar pra começar com jquery é http://demos.jquerymobile.com/1.3.2/intro/ Como está em ingles, a gente dá uma ajudinha aqui…

    1. mas o que é jquery?

      É uma biblioteca javascript https://learn.jquery.com/ Tem a versão pra sites normais e tem a versão para mobile Pra usar o jquery mobile vc tem que baixar ele e colocá-lo no seu projeto.

      Então os comandos abaixo vão baixar o jquery e instalá-lo no nosso projeto. Vc pode fazer manualmente também. 

      cd /home/wagner/wagnerdocri@gmail.com3/envs/env-dev/sources/aulas_de_cordova/projetos/jquerymob_menu/
wget --output-document jquerymobile.zip  https://jquerymobile.com/resources/download/jquery.mobile-1.4.5.zip                               
mkdir jquerymobile && unzip -d jquerymobile jquerymobile.zip

cp jquerymobile/demos/js/jquery.js www/js
cp jquerymobile/jquery.mobile-1.4.5.min.css www/css
cp jquerymobile/jquery.mobile-1.4.5.min.js www/js
cp -rf jquerymobile/images www

rm -r jquerymobile
